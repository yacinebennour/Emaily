# Emaily
An email scraper built with Puppeteer.

## Description

Scrape emails easily via Headless Puppeteer + the help of search queries. Simple project was built for fun.

## Getting Started

### Dependencies

* Puppeteer
* NodeJS

### Installing

* Simply run index.js!

### Executing program

```
npm i puppeteer
node index.js
```
